## 🚀 Featured Project

### 💰 [NexVault — Smart Finance & Wealth Management Platform](https://github.com/MOHANX-0001/nexvault)

> Full-stack FinTech app for tracking income, expenses, budgets, accounts & savings goals — with real-time charts and analytics.

```
React.js · Node.js · Express · PostgreSQL · Prisma ORM · JWT · Recharts · Vite
```

| What I built | How |
|---|---|
| JWT auth with **access + refresh token rotation** | `jsonwebtoken` + token invalidation on each rotation |
| PostgreSQL database with **8 Prisma models** | Designed full schema: User, Account, Transaction, Budget, Goal, Category, Notification, RefreshToken |
| **REST API with 7 route modules** + role-based auth middleware | Express.js with modular route structure |
| **Interactive dashboard** with real-time charts | Recharts — pie charts, bar charts, trend lines |
| **Notification system** with unread badge + slide-in panel | React state + CSS animations |
| **Dark / Light theme toggle** | CSS variables + localStorage persistence |
| **Graceful mock data fallback** when backend is unreachable | Axios interceptors + local mock dataset |
| Security: bcrypt, Helmet.js, rate limiting, CORS | Production-grade hardening |
